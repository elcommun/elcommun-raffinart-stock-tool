Raffinart 在庫更新ツール EL COMMUN

処理概要：「在庫CSV」のSKU末尾の品番コード（例: FIN-62604）をVol.3 / Vol.4の「品番」と照合し、総在庫数を更新します。
1

在庫データをアップロード Vol.3 / Vol.4 （xlsx）

在庫データのダウンロード元（Raffinart ストックリスト）
https://raffinart.jp/stock-list/ から 「INTERIOR STYLE Vol.3」 と 「INTERIOR STYLE Vol.4」 の在庫表（xlsx）をダウンロードしてください
📊

【在庫】raffinart_vol.3.xlsx

ドロップ または クリックして選択
📊

【在庫】raffinart_vol.4.xlsx

ドロップ または クリックして選択
2

在庫CSVをアップロード itemdata_*.csv

CSVのダウンロード方法（GoQ System）
「在庫連携画面」上部の 【CSV】 プルダウンで 「EL COMMUN 楽天市場店」 を選択 → 「すべて」 にチェック → 「CSV出力」 ボタンを押す
📄

itemdata_*.csv（SKU, 総在庫数）

ドロップ または クリックして選択
RULE

更新ルール

・SKU末尾の品番コード（英大文字＋ハイフン＋数字）でVol.3/Vol.4の「品番」と照合
・「在庫状況」が ※ または X → 総在庫数を 0 に更新
・それ以外（△・空欄） → 総在庫数を 10 に更新
・品番が 未マッチ → 総在庫数を 0 に更新し、下に一覧表示
・「入荷予定」が SOLD OUT → 出力から除外し、下に一覧表示
3

実行

✅ 更新完了！

⚠️ このあと必ず行うこと：
GoQ System「在庫連携画面」でCSVを取り込んだあと、「EL COMMUN 楽天市場店」で「差異商品」を検索し、「反映」させてください。
